# UXCode React Landing Page

This project is a React + Vite landing page for UXCode.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind (via CDN in `index.html`)
- Framer Motion
- Lucide React

## Run Locally
1. Install dependencies:
   `npm install`
2. Start development server:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Production Build
1. Build:
   `npm run build`
2. Preview:
   `npm run preview`

## Project Structure
- `App.tsx`: page composition
- `components/`: UI sections and shared components
- `constants.tsx`: content and configuration
- `types.ts`: shared types
